# Desktop Calendar

A weekly calendar grid widget for Noctalia that displays events positioned by time and duration.

![weekly calendar grid with events](https://github.com/Mic92/noctalia-plugins/releases/download/assets/desktop-calendar.png)

## Features
- Weekly calendar grid with hour rows and day columns
- Timed events with overlap layout
- All-day event row
- Week navigation (previous/next/today)
- Current time indicator
- Tooltips with event details

## Requirements
- Noctalia 4.5.0 or later
